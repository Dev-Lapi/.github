### 개요
**라피**는 **Unity 6**를 이용하여 개발한 픽셀아트 기반의 RPG 게임이며     
`🏆Beaver Rocks 2025🏆`의 **전시작으로 선정**되어 현재 STOVE Indie에서 다운로드 받을 수 있습니다.

</br>

라피는 꿈에서 길을 잃은 토끼가 꿈에서 탈출하기 위해 꿈 속을 모험하는 스토리 배경을 가지고 있습니다.</br>
플레이어는 꿈 속을 자유롭게 돌아다니며 잔잔한 BGM과 어우러지는 픽셀아트를 통해 몽환적인 느낌을 받습니다.
